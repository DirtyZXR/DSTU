Типы атрибута type элемента input

Тип Вид в браузере Описание
button Отображает кнопку, которая может быть нажата для выполнения определенного действия
checkbox Отображает флажок, который может быть выбран или не выбран
file Отображает поле для выбора файла на компьютере пользователя
hidden Отображает поле для ввода значения, которое не будет отображаться на странице
image При нажатии на кнопку, данные отпраляются на сервер.
password Отображает поле для ввода значения, которое не будет отображаться на странице
radio Отображает переключатель, который может быть выбран только один из нескольких вариантов
reset Сбрасывает все значения формы на значения по умолчанию
sumbit Отправляет данные формы на сервер
text Отображает текстовое поле для ввода одного слова или нескольких слов, разделенных пробелами
color Отображает поле для выбора цвета
date Отображает поле для выбора даты
datetime-local Отображает поле для выбора даты и времени
datetime Отображает поле для выбора даты и времени
email Отображает поле для ввода электронной почты
numner Отображает поле для ввода числа
range Отображает поле для выбора значения в указаном диапазоне
search Отображает поле для поиска
tel Отображает поле для ввода телефонного номера
time Отображает поле для выбора времени
url Отображает поле для ввода адреса URL
week Отображает поле для выбора недели
month Отображает поле для выбора месяца
